🔧 Installation Guide

Set up the complete open-source VLSI EDA toolchain on Ubuntu/Debian Linux.

Prerequisites

Make sure your system is up to date before installing:

# Update package lists
sudo apt update && sudo apt upgrade -y

# Install essential build tools
sudo apt install -y build-essential git cmake python3 python3-pip
OS: Ubuntu 20.04+ / Debian 11+
RAM: 4 GB minimum
Disk: ~5 GB free space
Shell: bash / zsh
1

Icarus Verilog

RTL Simulation — compile & run Verilog designs
# Install Icarus Verilog
sudo apt install -y iverilog

✅ Verify Installation

$ iverilog -V
# Should show: Icarus Verilog version 12.0 (or similar)

$ echo 'module test; initial $display("Hello Verilog!"); endmodule' > /tmp/test.v
$ iverilog -o /tmp/test /tmp/test.v && vvp /tmp/test
# Should print: Hello Verilog!

GTKWave

Covered

Code Coverage — analyze toggle, line, FSM coverage
# Clone the Covered repository
git clone https://github.com/hpretl/verilog-covered.git
cd verilog-covered

# Configure and build
./configure
make
sudo make install
⚠️ Build from source required. Covered is not in apt. If you encounter a TCL_DSTRING error during build, apply the patch from the project's GitHub issues page.

✅ Verify Installation

$ covered --version
# Should show: Covered - Verilog Code Coverage Analyzer
4

Yosys

Logic Synthesis — RTL to gate-level netlist
# Install Yosys
sudo apt install -y yosys

# (Optional) Install Graphviz for circuit diagrams
sudo apt install -y graphviz

✅ Verify Installation

$ yosys -V
# Should show: Yosys 0.33 (or similar)
5

OpenROAD

Physical Design Automation — floorplan, place, CTS, route
# Option A: Build from source (recommended for latest features)
git clone --recursive https://github.com/The-OpenROAD-Project/OpenROAD.git
cd OpenROAD

# Install dependencies (Ubuntu/Debian)
sudo ./etc/DependencyInstaller.sh

# Build
mkdir build && cd build
cmake ..
make -j$(nproc)
sudo make install
💡 Build takes ~30-60 minutes depending on your system. The DependencyInstaller.sh script handles all required libraries (Boost, TCL, SWIG, etc.). 
# The Nangate45 PDK test files are included with OpenROAD:
ls OpenROAD/test/Nangate45/
# → Nangate45_typ.lib  Nangate45_tech.lef  Nangate45_stdcell.lef  Nangate45.tracks

✅ Verify Installation

$ openroad -version
# Should show: OpenROAD v2.0-xxxx

# Quick test: start interactive mode
$ openroad
openroad> help
openroad> exit

🎉 You're All Set!

All 5 tools installed. You can now run the complete RTL → Simulation → Synthesis → PDA flow.
